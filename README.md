# Student Welfare Wing | SQAT Club, Daffodil International University

Welcome to the official web repository for the **Student Welfare Wing** of the Software Quality Assurance & Testing Club (SQAT), Daffodil International University (DIU). This website serves as a central hub for student support, academic peer-mentorship, volunteer drives, and community welfare initiatives.

🔗 **Live Website URL**: **[https://student-welfare-wing.vercel.app/](https://student-welfare-wing.vercel.app/)**

---

## ✨ Key Features

- **Premium Design System**: Tailored with glassmorphic cards, harmonized CSS variables, interactive mouse glow coordinates, and smooth entrance scroll reveals.
- **Dynamic Directory Search & Filter**: Real-time searching and select-filtering of members by name, department, and academic session.
- **Dynamic Lightbox Overlays**: Click-to-enlarge overlays on Patrons, Executive Board, Secretarial, and Wing Leadership cards that automatically render member names and designations.
- **SEO & Structured Metadata**: Fully optimized search engine indexing containing standard robots directives, sitemaps, and Schema.org (JSON-LD) structured cards.
- **High-Definition Gallery**: Visual grid of events, including welfare supplies campaigns, blood donation drives, exam prep circles, and winter clothing distribution campaigns.
- **Dark & Light Themes**: Standardized client-side dark/light toggle saving state across page reloads.

---

## 🛠️ Technology Stack

- **Structure**: HTML5 Semantic markup
- **Styling**: Vanilla CSS3 (Custom properties, grid systems, smooth transitions)
- **Logic**: Vanilla ES6 Javascript
- **Icons**: FontAwesome CDN

## 🚀 Running Locally

To run this project on your local machine, ensure you have **Node.js** installed, then execute:

```bash
# Clone the repository
git clone https://github.com/Shafiur0/student-welfare-wing.git

# Navigate into the project folder
cd student-welfare-wing

# Install dependencies (http-server for local dev)
npm install

# Start the local development server
npm run dev
```

The server will start locally at **[http://localhost:3000/](http://localhost:3000/)**.

## 🌎 Hosting & Deployments

This site is automatically deployed and hosted via **Vercel** with a Git integration pipeline:
